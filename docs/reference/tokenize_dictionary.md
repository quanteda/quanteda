# Separate dictionary values using tokenizer

Apply [`tokens()`](https://quanteda.io/reference/tokens.md) to
dictionary values to improve pattern matching.

## Usage

``` r
tokenize_dictionary(dictionary, ...)
```

## Arguments

- dictionary:

  a [dictionary](https://quanteda.io/reference/dictionary.md) object.

- ...:

  passed to [`tokens()`](https://quanteda.io/reference/tokens.md).
