# Internal functions for dfm objects

Internal function documentation for
[dfm](https://quanteda.io/reference/dfm.md) objects.

## Usage

``` r
# S4 method for class 'dfm,numeric'
Compare(e1, e2)
```

## Arguments

- e1:

  a [dfm](https://quanteda.io/reference/dfm.md)

- e2:

  a numeric value to compare with values in a dfm
