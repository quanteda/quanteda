# Check if an object is collocations

Function to check if an object is a collocations object, created by
[`quanteda.textstats::textstat_collocations()`](https://quanteda.io/reference/textstat_collocations.html).

## Usage

``` r
is.collocations(x)
```

## Arguments

- x:

  object to be checked

## Value

`TRUE` if the object is of class `collocations`, `FALSE` otherwise
